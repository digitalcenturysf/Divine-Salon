== Description ==

Divine Salon is a clean WordPress theme designed for all type of cosmetic websites such as salon, wellness and parlour etc. This WordPress theme comes with many great features like Visual Composer so you can create an unlimited number of beautiful pages. There are multiple gorgeous home pages, and over a dozen of inner pages for your website.
